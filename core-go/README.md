# Survey Core Module

Реализация главного модуля системы тестирования по сценарию

## Функционал
- Создание тестов и вопросов
- Прохождение тестов (попытки)
- Хранение и просмотр результатов
- Интеграция с модулем авторизации через JWT

## Технологии
- Go 1.21+
- Fiber (web framework)
- GORM (ORM для PostgreSQL)
- JWT (аутентификация)

## Запуск
go run .
